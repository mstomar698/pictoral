# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its code of
conduct. The full text is available at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating — contributing code, opening issues, commenting, or otherwise
engaging with the project — you agree to help keep this a welcoming, respectful,
and harassment-free community for everyone, regardless of background or identity.

## Reporting

If you experience or witness unacceptable behavior, please report it privately to
the maintainer via a
[GitHub security advisory](https://github.com/mstomar698/pictoral/security/advisories/new).
All reports are reviewed in good faith and handled in confidence.

## Enforcement

The maintainer is responsible for clarifying and enforcing the standards of
acceptable behavior and may take fair, appropriate corrective action in response
to any behavior deemed inappropriate, threatening, or harmful.
